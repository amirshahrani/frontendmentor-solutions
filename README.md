1. Clone the repository

   ```bash
   git clone https://github.com/amirshahrani/frontendmentor-solutions.git
   ```

2. Navigate to any challenge folder

   ```bash
   cd frontendmentor-solutions/profile-card
   ```

3. Open `index.html` in your browser
